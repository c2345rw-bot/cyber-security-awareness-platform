## 🎯 Problem Statement

Teenagers in Central Asia (Uzbekistan, Kazakhstan, Kyrgyzstan, Tajikistan, Turkmenistan) are **targeted daily** by:
- Telegram scams promising easy money
- Fake job offers requiring "registration fees"
- Cracked apps & games containing malware
- Phishing links disguised as official messages

Most cybersecurity resources are in English and focus on Western contexts — leaving local teens unprotected.

## 💡 Solution

**CyberSafe Teens** provides:
- **Region-specific threat examples** (not generic US/Europe scenarios)
- **Simple, actionable defense strategies** (no jargon)
- **Interactive phishing quiz** to test knowledge
- **Clear visual design** accessible to non-technical users

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎮 **Phishing Detection Quiz** | 5 real-world scenarios with instant explanations |
| 🔐 **Password Generator** *(coming soon)* | Create strong, unique passwords |
| 📱 **Social Media Security Guide** | Step-by-step privacy settings |
| 🧠 **Real Threat Examples** | Telegram scams, fake job offers, cracked apps |
